# Chef4U Catering Demo

This project is a premium landing page for **Chef4U Catering**, a premier catering service based in Ahmedabad with over 10 years of experience. The website showcases their services, menu highlights, and expertise in handling weddings, corporate events, and social gatherings.

## Features

- **Modern & Responsive Design**: A fully responsive layout that looks great on all devices.
- **Interactive Navigation**: Smooth scrolling to Home, About Us, and Services sections.
- **Online Inquiry Form**: A integrated contact form for bookings and inquiries.
- **Service Highlights**: Detailed sections for Corporate Catering, Social Catering, and Packed Meals.
- **WhatsApp Integration**: Quick-action button for direct communication with the catering team.
- **SEO Optimized**: Includes meta tags for better search engine visibility.

## Project Structure

- **index.html**: Main entry point containing the landing page structure.
- **style.css**: Custom CSS for styling and responsiveness.
- **assets/**: Contains project-specific images and static assets.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser to view the demo.
