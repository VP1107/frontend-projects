# Renav Messenger - TheMessenger Landing Page

[![GitHub](https://img.shields.io/badge/GitHub-vp1107-181717?logo=github)](https://github.com/vp1107/frontend-projects/tree/main/renav-messenger-website-clone)

A modern, responsive landing page for **TheMessenger**, an Official WhatsApp Business API solution by Renav Technologies. This page showcases the platform's capabilities for bulk broadcasting, chatbot automation, and CRM integration.
