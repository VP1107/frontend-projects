# Renav Messenger

This project is a landing page for "TheMessenger," a WhatsApp Business API solution by Renav Technologies. It showcases features like bulk broadcasting, chatbots, and CRM integration.

## Project Structure

- **index.html**: Main landing page.
- **css/**: Contains stylesheets (e.g., `style.css`).
- **js/**: Contains JavaScript files (currently empty).
- **assets/**: folder for images, logos, and other static media.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
